Converts Articulate Storyline SCORM files to IMSCC files that can be imported into Coursera.  Only handles the text content.

See `frontend/README.md` for information on building this.

# Usage

Automatically built and deployed here: https://kc0bfv.github.io/SCORMToIMSCC/

All content is in `index.html`, and this is a one-file application.
